# 西南三省经济数据可视化

西南三省（贵州、云南、广西）2025年经济数据可视化网页，包含367个地市区县的GDP与财政收入排名。

## 📊 数据概览

| 省份 | 地级市 | 区县 | 总计 |
|------|--------|------|------|
| 贵州省 | 9个 | 88个 | 97个 |
| 云南省 | 16个 | 129个 | 145个 |
| 广西 | 14个 | 111个 | 125个 |
| **合计** | **39个** | **328个** | **367个** |

## 📁 文件说明

- `index.html` - 首页：三省对比分析，367个地市区县GDP/财政收入完整排名
- `guizhou.html` - 贵州省：97个地市区县详细数据
- `yunnan.html` - 云南省：145个地市区县详细数据
- `guangxi.html` - 广西：125个地市区县详细数据
- `all_data.json` - 全部367个地市区县数据
- `city_data.json` - 39个地级市数据

## 📈 核心指标

### 三省总量对比

| 指标 | 贵州 | 云南 | 广西 |
|------|------|------|------|
| GDP | 22,562亿 | 31,636亿 | 27,616亿 |
| 财政收入 | 1,565亿 | 2,144亿 | 1,749亿 |
| 财政支出 | 4,670亿 | 5,961亿 | 5,229亿 |
| 债务余额 | 15,325亿 | 14,579亿 | 10,067亿 |
| 债务率 | 10.9倍 | 6.8倍 | 5.8倍 |

## 🎯 主要发现

1. **云南经济总量最大**：GDP 31,636亿元，位居三省之首
2. **贵州债务压力最大**：债务率高达10.9倍，需关注偿债风险
3. **广西财政最稳健**：债务率仅5.8倍，三省中最低
4. **强县经济突出**：仁怀市GDP 2,026亿元，超过多个地级市
5. **财政自给率偏低**：三省均在33-36%之间，依赖转移支付
